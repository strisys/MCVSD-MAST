# Arithmetic / Powers & Roots

### Problem 1

If $\sqrt[3]{216} = 10x$, what is the value of $5x + 4$?

### Problem 2

Simplify the following expression: $(x^5 y^8)^3$

### Problem 3

A factory produces $4^{-1}$ widgets per hour. If it operates for $4^2$ hours per day, how many widgets does it produce in a day?

### Problem 4

Simplify the expression $\frac{(a^2 b^3)^4}{a^5 b^{10}}$ and write the result using only positive exponents.

### Problem 5

Evaluate the expression $\sqrt{169} - \sqrt[3]{125} + 2^3$.

### Problem 6

If $5^{x+1} = 125$, what is the value of $x^2 + 3x$?

### Problem 7

A square tile has an area of 144 square inches. What is the perimeter of the tile, in inches?

### Problem 8

Write the following expression as a single integer: $(2^3)^2 \cdot 2^{-4}$.
