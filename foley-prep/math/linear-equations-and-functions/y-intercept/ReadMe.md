# Linear equations and functions

It looks like you're practicing finding the y-intercept of a linear function given a specific point.

The core concept for these problems is using the slope-intercept form, $f(x) = mx + b$, where $m$ is the slope and $b$ is the y-intercept. When you are given a point like $f(3) = 15$, it simply means that when $x = 3$, $y = 15$. You plug those values into the equation to solve for $b$.

Here are 5 similar practice problems for you:

------

### Practice Problems

**1.** If $f(2) = 10$, find the y-intercept of $f(x)$ where $f(x) = 3x + b$.

- A. $(0, 16)$
- B. $(0, 4)$
- C. $(2, 10)$
- D. $(0, -4)$

**2.** If $f(-4) = 2$, find the y-intercept of $f(x)$ where $f(x) = -1x + b$.

- A. $(0, -2)$
- B. $(0, 6)$
- C. $(0, 2)$
- D. $(0, -6)$

**3.** If $f(5) = -3$, find the y-intercept of $f(x)$ where $f(x) = 2x + b$.

- A. $(0, -13)$
- B. $(0, 7)$
- C. $(0, 13)$
- D. $(0, -8)$

**4.** If $f(6) = 12$, find the y-intercept of $f(x)$ where $f(x) = \frac{1}{2}x + b$.

- A. $(0, 15)$
- B. $(0, 6)$
- C. $(0, 9)$
- D. $(0, 0)$

**5.** If $f(-3) = -11$, find the y-intercept of $f(x)$ where $f(x) = 4x + b$.

- A. $(0, 1)$
- B. $(0, -23)$
- C. $(0, -11)$
- D. $(0, -1)$
