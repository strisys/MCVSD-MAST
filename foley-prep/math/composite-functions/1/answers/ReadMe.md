# Composite Functions 



To solve the problem in your image, you need to use **composite functions**. The goal is to evaluate the inner function first, and then use that result as the input for the outer function.

Here are three similar problems designed to help you practice this substitution and evaluation process.

------

## Practice Problems

### Problem 1: Linear and Quadratic Mix

If $f(x) = 3x - 4$ and $g(x) = \frac{x^2 - 10}{x + 2}$, find the value of $g(f(3))$.

### Problem 2: Radical and Rational Mix

If $h(x) = \sqrt{x + 5}$ and $j(x) = \frac{2x^2 - 4}{x - 1}$, find the value of $j(h(11))$.

### Problem 3: Higher Degree Polynomials

If $p(x) = x^2 - 1$ and $q(x) = \frac{x^2 - 5x + 6}{x - 4}$, find the value of $q(p(3))$.

------

## Answer Key and Step-by-Step Solutions

### Solution 1:

1. **Find $f(3)$:** $f(3) = 3(3) - 4 = 9 - 4 = 5$.

2. Find $g(5)$: Substitute $5$ into $g(x)$:

   

   $$g(5) = \frac{5^2 - 10}{5 + 2} = \frac{25 - 10}{7} = \frac{15}{7}$$

- **Answer:** $\frac{15}{7}$

### Solution 2:

1. **Find $h(11)$:** $h(11) = \sqrt{11 + 5} = \sqrt{16} = 4$.

2. Find $j(4)$: Substitute $4$ into $j(x)$:

   

   $$j(4) = \frac{2(4)^2 - 4}{4 - 1} = \frac{2(16) - 4}{3} = \frac{32 - 4}{3} = \frac{28}{3}$$

- **Answer:** $\frac{28}{3}$

### Solution 3:

1. **Find $p(3)$:** $p(3) = (3)^2 - 1 = 9 - 1 = 8$.

2. Find $q(8)$: Substitute $8$ into $q(x)$:

   

   $$q(8) = \frac{8^2 - 5(8) + 6}{8 - 4} = \frac{64 - 40 + 6}{4} = \frac{30}{4} = 7.5$$

- **Answer:** $7.5$ (or $\frac{15}{2}$)
